# Release Checklist

## Before Launch

- Domain confirmed
- Hosting target confirmed
- Environment variables documented, if any
- Analytics configured, if needed
- Final copy approved
- Images approved
- Legal/age-related note approved

## Launch

- Deploy site
- Verify production URL
- Verify map and social links
- Verify mobile view on a real device
