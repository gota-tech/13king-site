# Technical Approach

## Initial Recommendation

Use a dependency-free static landing page unless the project later needs a CMS, reservation system, or frequent menu updates.

## Selected Stack

- HTML
- CSS
- JavaScript
- No framework for the first version

## Future Candidate Stack

- Astro/Next.js if deployment, routing, CMS, or component reuse becomes important.
