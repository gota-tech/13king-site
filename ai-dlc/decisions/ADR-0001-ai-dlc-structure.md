# ADR-0001: AI-DLC Structure

## Status

Accepted

## Context

The project needs a lightweight structure for AI-assisted development before implementation starts.

## Decision

Use an `ai-dlc/` folder with phase-based subfolders for charter, requirements, design, planning, implementation, validation, release, and operations.

## Consequences

- The project can keep prompts, assumptions, decisions, and validation notes close to the codebase.
- The structure remains lightweight enough for a small landing page.
- More formal artifacts can be added later if the site grows.
